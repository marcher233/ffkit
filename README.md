饭否小工具

“饭否小工具”是一些可运行在SAE（ http://sae.sina.com.cn/ ）平台上的PHP小程序集合。

包含组件：
- 饭友生日 /birthday
- 时间线清理工具 /timeline_cleaner
- 马闹钟 /alarma
- 饭否5周年自曝活动页 /5
- 中枪 /imhit
- 五笔机器人 /ggtt
- 头像仓库 /avatar
- 数据表结构 /db

非专业人员，代码相当混乱，请见谅。 所有代码均开源，无使用限制。欢迎fork。
